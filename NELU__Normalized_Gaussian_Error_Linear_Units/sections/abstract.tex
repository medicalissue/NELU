\begin{abstract}
Smooth self-gated activations---those of the form $z_i \cdot g(z_i)$, including GELU and SiLU---replaced ReLU's binary gate with graded, differentiable alternatives.
We show that this trade gained smoothness but lost a property ReLU had: scale invariance.
For any non-constant smooth gate $g$, rescaling the pre-activations changes not only the output magnitude but also its \emph{direction}, entangling downstream feature selection with absolute scale.
We propose NELU (Normalized Gaussian Error Linear Unit), which restores scale invariance within the self-gated form:
$\mathrm{NELU}(\mathbf{z})_i = z_i\,\Phi\!\bigl(z_i / \mathrm{sg}[\mathrm{rms}(\mathbf{z})]\bigr)$.
NELU's output direction depends only on the normalized pattern ($\mathrm{NELU}(\alpha\mathbf{z}) = \alpha\,\mathrm{NELU}(\mathbf{z})$); scale is not discarded but redirected to the output magnitude.
No parameters are added; NELU is a drop-in GELU replacement.
Across CNNs on CIFAR-10/100, ViTs on ImageNet, and GPT-2 language models up to 774M parameters, NELU matches or improves GELU on clean benchmarks.
On corruption benchmarks---where pre-activation scale distributions shift---NELU's gains are largest, consistent with its scale-invariant design.
\end{abstract}
